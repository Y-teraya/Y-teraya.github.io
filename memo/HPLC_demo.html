HPLC の使い方

高速液体クロマトグラフィー

High Performance Liquid Chromatography

HPLCとは？

  • 液体試料中の成分を分離・定量する分析装置
  • 医薬品、食品、環境分析など幅広い分野で使用
  • 高い分離能力と感度を持つ
  • 微量成分の分析が可能

HPLCの基本構成

  • ポンプ: 移動相を送液
  • インジェクター: 試料を注入
  • カラム: 成分を分離
  • 検出器: 成分を検出
  • データ処理装置: 結果を記録・解析

使用前の準備

  1. 移動相の調製とフィルター濾過
  2. 脱気処理（超音波またはヘリウムバブリング）
  3. カラムの選択と取り付け
  4. システムの漏れチェック
⚠️ 移動相は必ずHPLCグレードの溶媒を使用

起動手順 (1)

1電源投入

  • 検出器 → ポンプ → データ処理装置の順に電源ON
  • 検出器のランプを30分以上ウォームアップ

2移動相の設定

  • 使用する溶媒ラインを確認
  • 流量と組成を設定

起動手順 (2)

3プライミング

  • 各溶媒ラインをパージ（10mL以上）
  • 気泡を完全に除去

4カラム平衡化

  • 設定流量で移動相を流す
  • ベースラインが安定するまで待機（30分以上）

試料注入

  1. 試料の前処理: フィルター濾過（0.45μm以下）
  2. バイアルに充填: 適量をバイアルに入れる
  3. オートサンプラーにセット: 位置を確認
  4. メソッドの設定: 注入量、分析時間を入力
  5. 測定開始: スタートボタンを押す

データ取得と解析

  • クロマトグラムの確認: ピーク形状をチェック
  • 保持時間: 各ピークの溶出時間
  • ピーク面積/高さ: 定量に使用
  • 分離度: ピーク間の分離状態を評価

終了手順

  1. カラムを保存溶媒で洗浄（カラムの種類に応じて）
  2. 流量を下げてから停止（0.1-0.2 mL/min）
  3. 検出器のランプを消灯
  4. データを保存・バックアップ
  5. 装置の電源OFF（データ処理装置 → ポンプ → 検出器）

よくあるトラブル

症状 原因 対策
圧力異常上昇 カラム詰まり カラム逆洗、交換
ピーク分離不良 カラム劣化 カラム交換、条件最適化
ベースライン不安定 気泡混入 脱気、プライミング
ピーク形状不良 試料過負荷 注入量減少

日常メンテナンス

  • 毎日: 移動相ボトルの確認、廃液チェック
  • 毎週: フィルター・インラインフィルターの交換
  • 毎月: ポンプシールの点検、カラムの洗浄
  • 定期: ランプ交換、システム適格性評価

安全上の注意

  • 有機溶媒は必ずドラフト内で取り扱う
  • 高圧システムのため、配管の緩み注意
  • 廃液は適切に処理する
  • 保護具（手袋、保護メガネ）を着用
  • 緊急時の対応手順を確認しておく

まとめ

  • 適切な準備と手順で正確な分析が可能
  • 日常メンテナンスで装置を長持ちさせる
  • トラブル時は冷静に原因を特定
  • 安全第一で操作する
